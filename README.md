Max OSX
1. Install NodeJS
2. Install Cordova ```sudo npm install -g cordova```
3. Install Ionic ```sudo npm install -g ionic```
4. Install Bower ```sudo npm install -g bower```
5. Clone Git Repo
6. ```ionic state restore```
7. ```bower install```
8. ```ionic emulate ios``` or ```ionic run ios```
